# epoch's Tech Life

记录技术探索与生活思考的个人博客。

## 技术栈

- **框架**: [Astro](https://astro.build/)
- **样式**: [Tailwind CSS](https://tailwindcss.com/)
- **内容**: Markdown / MDX
- **搜索**: [Pagefind](https://pagefind.app/)
- **主题**: 基于 [AstroPaper](https://github.com/satnaing/astro-paper) 改造

## 本地开发

```bash
pnpm install
pnpm dev
```

## 常用命令

| 命令             | 说明              |
| :--------------- | :---------------- |
| `pnpm dev`       | 启动开发服务器     |
| `pnpm build`     | 构建生产版本       |
| `pnpm preview`   | 本地预览构建结果   |

## 配置

站点配置在 `astro-paper.config.ts`，主题与构架配置在 `astro.config.ts`。
